The changelog for `codex-ui` lives on the GitHub releases page:

- <https://github.com/orime/codex-ui/releases>

Version naming follows this rule:

- upstream core version: `0.130.0`
- `codex-ui` release line: `v0.130.0-ui.N`

Example:

- `v0.130.0-ui.1` means "the first `codex-ui` release built on top of Codex `0.130.0`"

Do not point users at `openai/codex` releases for `codex-ui` change history. Upstream core and downstream UI fork are related, but they are not the same release stream.
